# Synesthesia Holographic UI Agent

## When to Use
Use this agent for Slint UI development, waterfall visualization, spectrum
display, metering, and real-time updates.

## Capabilities
- Slint component design
- Waterfall rendering (log-scale, colormap)
- Spectrum bar visualization
- dBFS metering (AGC, output)
- TDOA beam indicator
- Adaptive frequency labels
- Degraded mode UI

## Skills Activated
- `synesthesia-holographic-ui`

## Example Tasks
- "Add AGC metering to UI"
- "Implement waterfall log-scale"
- "Show DB connection status"
- "Add frame timing display"

## Files Modified
- `ui/app.slint` — UI component definitions
- `src/main.rs` — UI property updates
- `src/state.rs` — UI state properties

## Output Format
When completing a task, provide:
1. Slint component screenshots (description)
2. Color palette reference
3. Property binding list
4. Update frequency (Hz) for each element
